# 人事管理系统 java625

## 项目概述

人事管理系统是基于 Spring Boot 和 Vue 的前后端分离架构，旨在提供高效、易用的人事管理解决方案。

## 技术栈

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL 5.7 & 8
- 运行环境：Node.js 14
- 构建工具：Maven

## 功能模块

系统包含但不限于以下功能模块：

- 员工信息管理
- 考勤管理
- 薪资管理
- 招聘管理
- 培训管理

## 系统角色

- 系统管理员
- 人事专员
- 普通员工

## 运行环境

- JDK 1.8 或更高版本
- Maven 3.6 或更高版本
- MySQL 8.0（推荐）
- Node.js 14

## 部署步骤

1. 环境准备：确保所有依赖环境安装完成。
2. 数据库搭建：根据提供的 SQL 脚本初始化数据库。
3. 后端部署：使用 Maven 构建项目，运行主程序。
4. 前端部署：在 Node.js 环境下构建前端项目，并部署到 Web 服务器。

## 目录结构

```
人事管理系统
├── backend                后端代码
├── frontend               前端代码
└── sql                    数据库初始化脚本
```

## 核心亮点

- 前后端分离架构，易于维护和扩展。
- 支持多种数据库版本，灵活适应不同环境需求。
